⚠ ต้องตั้งค่า API ก่อนใช้งาน

ระบบยังไม่ได้รับ SCRYDEX_API_KEY และ SCRYDEX_TEAM_ID

  1. ไปที่ Vercel Dashboard → Project → Settings → Environment Variables
  2. เพิ่ม 2 ตัว: SCRYDEX_API_KEY และ SCRYDEX_TEAM_ID (เอามาจาก Scrydex Account Hub)
  3. กด Redeploy เพื่อให้ตัวแปรใหม่มีผล

Local dev: สร้างไฟล์ .env.local ตามตัวอย่างใน .env.local.example แล้วรัน npx vercel dev
